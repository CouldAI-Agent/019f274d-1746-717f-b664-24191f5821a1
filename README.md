# Calculator App

A beautiful, fully functional calculator application built with Flutter.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division).
- Clean, responsive user interface.
- Handles complex mathematical expressions smoothly.

## Tech Stack
- Flutter
- Dart
- `math_expressions` package for robust equation evaluation.

## Setup & Run
1. Ensure Flutter is installed and set up on your machine.
2. Run `flutter pub get` to fetch dependencies.
3. Start the app using `flutter run`.
